## 3. 适配层设计（适配器模式）

```python
class BaseLLM:
    def chat(self, messages, **kwargs):
        raise NotImplementedError

class OpenAILLM(BaseLLM): ...
class QwenLLM(BaseLLM): ...
class DeepSeekLLM(BaseLLM): ...

# 一行切换
llm = get_llm("qwen")
```

**核心价值**：业务代码不依赖具体模型，切换只改配置
**加分项**：fallback 机制 — 主模型挂了自动切备用模型
