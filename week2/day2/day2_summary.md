# 第 2 周 Day 2 学习总结：深度学习基础

> **核心目标：理解神经网络的训练原理，掌握前向传播、反向传播、梯度下降等核心概念。**

---

## 1. 深度学习 vs 传统机器学习

**核心区别**：
- **传统机器学习**：需要人工设计特征
- **深度学习**：模型自动学习特征（通过多层神经网络）

**为什么叫"深度"**：
- 深度 = 神经网络的层数多（几十层甚至上百层）
- 层数越多 → 能学习的特征越复杂

---

## 2. 神经网络结构

```
输入层 → 隐藏层（多层）→ 输出层
  ↓         ↓              ↓
 数据    特征提取         结果
```

**例子：图像分类**
```
输入层：像素值
隐藏层 1：识别边缘、线条
隐藏层 2：识别眼睛、耳朵
隐藏层 3：组合成完整物体
输出层：分类结果
```

---

## 3. 神经网络训练流程

```
1️⃣ 前向传播（Forward Propagation）
   输入数据 → 经过神经网络 → 得到预测

2️⃣ 计算损失（Loss Function）
   预测 vs 真实标签 → 计算差距
   
3️⃣ 反向传播（Backpropagation）
   从输出层往回算 → 计算每个参数的梯度
   
4️⃣ 梯度下降（Gradient Descent）
   参数 = 参数 - 学习率 × 梯度
   
5️⃣ 重复 1-4，直到损失足够小
```

---

## 4. 核心概念详解

### 4.1 损失函数

**作用**：衡量"预测有多错"

**常见损失函数**：
- 分类任务：交叉熵（Cross Entropy）
- 回归任务：均方误差（MSE）

**原则**：
- 预测越准确 → 损失越小
- 预测越离谱 → 损失越大

### 4.2 梯度下降

**下山比喻**：
- 山的高度 = 损失大小
- 你的位置 = 当前参数
- 坡度 = 梯度（告诉你往哪调整）
- 步长 = 学习率

**学习率选择**：
- 太大：快但不稳定，可能震荡
- 太小：慢但稳定，能找到精确解
- 常用技巧：学习率衰减（训练后期减小学习率）

---

## 5. 防止过拟合的技术

### 5.1 Dropout

**原理**：训练时随机关闭一些神经元

**效果**：
- 不过分依赖某些特征
- 强迫模型学会多种特征组合
- 提高泛化能力

**在反向传播阶段起作用**：被关闭的神经元不计算梯度

### 5.2 正则化（L1/L2）

**原理**：在损失函数中惩罚大参数

```
总损失 = 预测损失 + λ × 参数大小
```

**效果**：
- 防止参数过大
- 让模型更平滑、更稳健

**在反向传播阶段起作用**：影响梯度计算和参数更新

### 5.3 Early Stopping

**原理**：监控验证集损失，开始上升就停止训练

**效果**：防止过度训练

**注意**：在训练循环外层起作用，不在反向传播内部

## 6. 面试考点

| 面试问法 | 怎么答 |
|---------|--------|
| "什么是反向传播？" | "从输出层往回计算每个参数对损失的贡献（梯度），用于更新参数" |
| "梯度下降是什么？" | "根据梯度方向调整参数，让损失不断下降。就像下山，往最陡的方向走" |
| "怎么防止过拟合？" | "Dropout（随机关闭神经元）、L2正则化（惩罚大参数）、Early Stopping（提前停止）、数据增强" |
| "学习率太大会怎样？" | "下降快但不稳定，可能在最优点附近震荡。应该用学习率衰减" |

---

## 7. 关键理解

**神经网络训练 = 不断试错 + 调整**

1. 模型瞎猜 → 计算错得多离谱 → 分析哪里错了 → 调整参数
2. 重复上述过程几千次
3. 最终学会正确的判断规律

**防止过拟合 = 不让模型"死记硬背"**

- Dropout：不依赖某些特征
- 正则化：不要太自信
- Early Stopping：适可而止
