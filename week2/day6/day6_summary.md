# 第 2 周 Day 6 学习总结：Tokenizer 原理与 Embedding 向量空间

> **核心目标：理解分词器（Tokenizer）的工作原理和 Embedding 向量空间的意义。**

---

## 1. Tokenizer 是什么？

模型不认识"文字"，只认识**数字**。Tokenizer 是把文字转成数字序列的"翻译官"。

**前端类比**：`Tokenizer ≈ JSON.parse()`，把字符串变成可处理的结构。

---

## 2. 三种分词方案对比

| 方案 | 粒度 | 优缺点 |
|------|------|--------|
| 字级别 | 按字切 | 词表小，但丢失语义 |
| 词级别 | 按词切 | 语义完整，但词表爆炸（OOV） |
| **子词级别（BPE）** | 介于字与词之间 | ✅ 平衡词表大小和语义 |

---

## 3. BPE 算法核心原理

**核心思想**：从最小粒度开始，不断合并最高频的相邻对。

```
训练语料："low lower lowest"

Step 1: 按字母拆 → l, o, w, e, r, s, t
Step 2: 统计相邻对频率 → (l, o) 出现 3 次
Step 3: 合并最高频 → l + o → "lo"
Step 4: 重复直到达到目标词表大小
```

**结果**：高频词保留完整 token，低频词拆成子词。

**面试考点**：
- **问法**：「解释 BPE 分词算法的过程」
- **加分项**：GPT 使用 **Byte-level BPE**，直接在字节上操作，天然支持多语言

---

## 4. 主流 Tokenizer 对比

| 算法 | 使用模型 | 核心区别 |
|------|---------|---------|
| BPE | GPT 系列 | 按频率合并 |
| WordPiece | BERT | 按似然概率选择合并 |
| SentencePiece | T5/LLaMA | 语言无关，空格也当字符 |

---

## 5. Embedding 向量空间

### 为什么需要 Embedding？

Token ID 只是编号（如 3871），没有语义信息。Embedding 把编号变成**有语义的向量**。

**前端类比**：RGB 色值就是一种 Embedding！
```
红色 = [255, 0, 0]
橙色 = [255, 165, 0]（和红色"近"）
蓝色 = [0, 0, 255]（和红色"远"）

同理：
"猫" = [0.23, -0.45, 0.87, ...]
"狗" = [0.25, -0.42, 0.83, ...]（和"猫"近！）
"飞机" = [-0.67, 0.91, -0.12, ...]（和"猫"远）
```

### 向量距离 = 语义相似度

```
余弦相似度：
cos("猫", "狗") = 0.92    ← 相似
cos("猫", "飞机") = 0.12  ← 不相似
```

**这就是 RAG 语义检索的基础！**

### 面试考点

- **问法**：「Embedding 和 One-Hot 编码的区别？」
- **答法**：Embedding 是低维稠密向量（768维），能捕捉语义关系；One-Hot 是高维稀疏向量（50000维），无法表达语义
- **加分项**：大模型 Embedding 是上下文相关的，同一词在不同句子中向量不同

## 6. 代码实操验证

```python
import tiktoken
enc = tiktoken.get_encoding("cl100k_base")  # GPT-4 tokenizer

# 英文: "Hello, how are you?" → 6 个 token
# 中文: "你好，你怎么样？"   → 9 个 token

# 关键发现：中文 token 约是英文的 1.5 倍！
# 原因：BPE 训练语料以英文为主，中文字被拆成更多子 token
```

**运行结果**：
```
英文: Hello | , | how | are | you | ?
中文: 你 | 好 | ， | 你 | (怎被拆成2个子token) | 么 | 样 | ？
```

---

## 7. 避坑指南

1. **BPE 是训练阶段学的**，推理时使用已训练好的词表
2. **中英文 token 成本不同**：1 中文字 ≈ 2-3 个 token（API 计费影响）
3. **Embedding ≠ Word2Vec**：大模型 Embedding 是上下文相关的
4. **不能用 token ID 算语义距离**，必须先转 Embedding 向量
